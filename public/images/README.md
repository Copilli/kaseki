# Carpeta de Imágenes

Coloca aquí las imágenes que quieras usar en la aplicación.

## Imágenes necesarias:

### Para el NavBar:
- `logo.png` - Logo del proyecto (recomendado: 40-50px de altura)

### Para About Us:
- `hero.jpg` - Imagen grande para la tarjeta principal (recomendado: 800x600px)
- `regina.jpg` - Foto de Regina Garcia
- `ana.jpg` - Foto de Ana Sandoval
- `diego.jpg` - Foto de Diego Velazquez
- `gael.jpg` - Foto de Gael Casas
- `yareni.jpg` - Foto de Yareni Saavedra

## Uso en el código:

Una vez que coloques las imágenes aquí, usa rutas como:
- `/images/logo.png`
- `/images/hero.jpg`
- `/images/regina.jpg`
